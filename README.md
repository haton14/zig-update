# zig-update

Zig updator

## Usage

```
$ usage: zig-update [type] [path]
  aarch64-linux
  aarch64-macos
  x86_64-freebsd
  x86_64-linux
  x86_64-macos
  x86_64-windows
```

## Installation

```
go install github.com/mattn/zig-update@latest
```
